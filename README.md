# ✨ Overview

ProjectFlow Pro is a modern project management platform built with
**Pure Java HttpServer**, featuring:

-   📊 Live Dashboard
-   📁 Project Management
-   📋 Drag & Drop Kanban
-   🤖 AI Assistant
-   👥 Team Management
-   🌙 Dark Mode
-   🔒 Role Based Authentication
-   📈 Analytics
